试验五

目的：学习和掌握回溯算法

问题描述：0-1背包问题（0-1 Knapsack Problem）
    选择问题：给定n种物品和一背包。物品i的重量是wi，其价值为vi，背包的容量为C。
              问应如何选择装入背包的物品，使得装入背包中物品的总价值最大?
实例：v1 = 1,   v2 = 3,   v3 = 5,  v4 = 9,
      w1 = 2,   w2 = 3,  w3 = 4,  w4 = 7,
      背包容量：C=10
求解：放入物品2,4, 价值为12， 容量为10
        

根据上述分析，采用回溯法完成如下要求：
（1）用递归形式实现，程序实现2种输出：输出一个可能的装入方案，输出所有可能的装入方案；
（2）用非递归形式实现，程序实现2种输出：输出一个可能的装入方案，输出所有可能的装入方案；
（3）采用20个以上的物品进行5组不同的实验（数据自行定义），分析递归和非递归形式2种实现程序实际的运行时间差异。
     
   

要求：
1、深入分析和理解问题，了解和学习回溯算法；
2、设计算法，给出算法的伪代码；
