试验七

目的：学习和掌握分支限界算法

问题描述：0-1背包问题（0-1 Knapsack Problem）
    选择问题：给定n种物品和一背包。物品i的重量是wi，其价值为vi，背包的容量为C。
              问应如何选择装入背包的物品，使得装入背包中物品的总价值最大?
实例：v1 = 1,   v2 = 3,   v3 = 5,  v4 = 9,
      w1 = 2,   w2 = 3,  w3 = 4,  w4 = 7,
      背包容量：C=10
求解：放入物品2,4, 价值为12， 容量为10
        
      

根据上述分析，采用分支限界法完成如下要求：
（1）用队列（优先队列）形式实现分支限界法；
 (2) 采用贪心法作为界限函数的估计，对于部分解，也可以采用贪心法求解上界；
（3）采用20个以上的物品进行5组不同的实验（数据自行定义），采用相同的实验条件和数据，对比分支限界法和回溯法的效率差异（实验5）。
